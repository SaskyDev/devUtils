Query String Parser

Extract and analyze parameters from a URL query string. 







What is a query string?




A query string is the part of a URL that contains parameters after the question mark.
These parameters are commonly used to send data to web applications.





How to use this tool



    


  1. Paste a query string.
    2. 

  2. Click parse.
    3. 

  3. The parameters will appear below.
    4. 






Example



?user=alex&id=5



Example output:

id: 5
