# DevUtils

Simple developer tools that run directly in the browser.

Tools included:

* JSON Formatter
* Regex Tester
* Base64 Encoder / Decoder
* Timestamp Converter
* UUID Generator

Goal of the project:

Create a large collection of lightweight developer tools focused on usability and speed.

All tools work **client-side (JavaScript)** and require no backend.
