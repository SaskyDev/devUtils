# DevUtils

Simple developer tools that run directly in the browser.

Live site: https://devutilskit.com

Domain status: custom domain purchased and configured (`devutilskit.com`).

Tools included:

* JSON Formatter
* Regex Tester
* Base64 Encoder / Decoder
* Timestamp Converter
* UUID Generator
* JWT Encoder / Decoder
* Hash Generator
* Date Formatter
* CSV to JSON
* JSON to CSV
* URL Parser
* Query String Parser
* Color Converter

Goal of the project:

Create a large collection of lightweight developer tools focused on usability and speed.

All tools work **client-side (JavaScript)** and require no backend.

Recent updates:

* SEO improvements in key tool pages (titles, descriptions, canonical links)
* Expanded `sitemap.xml` priorities for better indexing signals
* UX feedback improvements in password and URL extractor flows
